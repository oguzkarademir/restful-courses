# Project Restful courses

This project to follow the [Build RESTful APIs with Node and Express](https://www.youtube.com/watch?v=pKd0Rpw7O48) by Mosh.

To implement this project use:
- Express framework
- JSON schemas
- Postman for testing
- _environmental variables
- nodemon
